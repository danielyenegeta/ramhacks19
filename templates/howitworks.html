Our Project

Stock5 allows users to assess the health of stock symbols. It does this by first scraping economical sites to retreive information on how the economy is doing (GDP, Unemployment Rate), then it creates a forecast using FBProphet from that data. In the end, the user gets a report on the forecast and how the stock is doing.

    Technologies used:

  • AWS
  • FBProphet
  • Flask
  • Bootstrap4

    What we could've done better

  • More Analysis on the Stock symbol
  • Used a database to hold all scraping information.
  • Hosted the site for a much easier demo
